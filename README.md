LLVMを用いたBrainf**kコンパイラ
## コマンドラインオプション

本プログラムは以下のコマンドラインオプションをサポートしています。

| オプション | パラメータ | 説明 |
| :--- | :--- | :--- |
| `-emit-llvm` | なし | LLVM IR を出力します。 |
| `-jit` | なし | JIT コンパイラを使用してその場でコードを実行します。 |
| `-compile-only` | なし | コンパイル処理のみを行います |
| `-verbose` | なし | 詳細モードを有効にします。 |
| `-memsize` | `<サイズ>` | 実行時に確保するメモリのサイズを数値で指定します。|
| `-o` | `<ファイル名>` | 出力するファイルのパスや名前を指定します。 |
